## Run the project

Install all dependencies

```bash
  yarn
```

### Set the `.env`

#### URL for the api

```bash
  REACT_APP_API_BASE_URL
```

#### URL for the front-end for tests with cypress

```bash
  APP_URL
```

Start the server

```bash
  yarn start
```

## Run the tests with cypress

**You need to run the back-end and front-end to run the tests**

### Start cypress

```bash
  yarn run cypress open
```

### Run the tests
